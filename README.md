# MaltaJS Elm event

## Requirements

 - NodeJs >= 4.3.2
 - Npm >= 3.1.0
 - MongoDB

## Instructions

$ cd devtools

$ npm install

$ elm package install -y

$ gulp build

$ gulp server

Navigate to `localhost:3000`
